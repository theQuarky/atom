Orbital — Electron Configuration Engine

This repository contains a TypeScript electron-configuration engine used by the Orbital visualizer.

Usage

Import the function and call getElectronConfig(z) with the atomic number (1–118).

import { getElectronConfig } from './src/electronConfig';

const fe = getElectronConfig(26);
console.log(fe);

Notes

  • Subshell fill order is generated programmatically (sorted by n+l, then n).
  • Known Aufbau exceptions (Cr, Cu, Pd, etc.) are hardcoded as overrides as requested.
  • The engine validates that the returned subshell electron counts sum to Z.

Tests

Vitest tests are included under the tests/ directory. Run with:

npm install
npm install -D vitest
npx vitest
